app.services BudgetService

Service class responsible for managing user budgets. Uses an in-memory ArrayList<Budget> cache (loaded lazily from file) for fast access. Persists changes to budgets.txt via FileDatabase.

⚙️ SOLID Notes

S – only budget business logic.  O – new alert thresholds can be added without changing callers.  D – depends on FileDatabase abstraction for I/O.
