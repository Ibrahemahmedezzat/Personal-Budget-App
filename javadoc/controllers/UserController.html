app.controllers UserController

Controller class responsible for handling user-related requests such as signup, login, and budget updates. Acts as a bridge between the UI presentation layer and the UserService.

This class is part of the MVC Controller layer.

⚙️ SOLID Notes

S – handles only user-domain requests.  D – UI depends on this controller abstraction, not directly on UserService.
