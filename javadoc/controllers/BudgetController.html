app.controllers BudgetController

Controller class responsible for handling budget-related requests. Bridges the UI and BudgetService. BudgetService stores budgets in an in-memory ArrayList and persists to file.

⚙️ SOLID Notes

S – ONLY budget domain.  D – UI depends on BudgetController abstraction, not BudgetService directly.
