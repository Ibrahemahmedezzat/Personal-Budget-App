java.lang.Object → app.data.FileDatabase

app.data FileDatabase

Utility class (Data Access Object) for flat-file persistence. All methods are static. Reads and writes comma-delimited lines in the data/ directory.

⚙️ SOLID Notes

S – only file I/O, no business logic.  O – switching to a database requires changing only this class.  D – all services depend on this abstraction, not on raw file APIs.
