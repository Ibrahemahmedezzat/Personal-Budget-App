Personal Budget Management System 💰📊

A full-featured desktop financial management system designed to help users take control of their personal finances through structured tracking, budgeting, and analytics.

Built with Java and JavaFX, the system provides an intuitive and responsive UI that allows users to manage income, expenses, budgets, and savings goals in a seamless workflow. It also generates insightful financial reports to support better decision-making.

✨ Key Features:
• Income & Expense tracking with categories  
• Monthly budget creation and monitoring  
• Savings goal management with progress tracking  
• Financial reports with analytical insights  
• Clean and modern JavaFX UI  
• Persistent data storage using database integration  

🧠 Technical Highlights:
• Object-Oriented Design (OOP)  
• MVC Architecture  
• SOLID Principles  
• Modular and scalable code structure  
• JavaFX for GUI development  
• Database integration for data persistence  

📚 Academic Context:
Developed as part of CS251 – Introduction to Software Engineering, Cairo University.
